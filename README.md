
# BucharestVPS Hugo Starter

Static, zero-Node build. AlpineJS for filters and theme toggle.

## Build
- Install Hugo (extended).
- `hugo --gc --minify`

## Cloudflare Pages
- Framework preset: Hugo
- Build command: `hugo --gc --minify`
- Environment var: `HUGO_VERSION=0.128.2`

## Customize
- Colors in `assets/css/tokens.css`
- Data in `data/` (plans, deals)
- Hosts and Datacenters in `content/hosts/*` and `content/datacenters/*`
- Affiliate redirects in `/_redirects`
